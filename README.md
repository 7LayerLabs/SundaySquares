## ‚ú® Key Features

- **Direct Interface**: No complex jargon. Just "Enter Pool" or "New Pool".
- **Dynamic Grid**: 10x10 interactive grid for square selection.
- **Admin Dashboard**: Secure control center for managing teams, scores, and verifying payments.
- **Axis Randomization**: Generate numbers only after the grid is locked to ensure fairness.
- **Real-time Winner Detection**: Automatically highlights the winning square based on the current score.
- **Payment Integration**: Support for Venmo, Cash App, and Cash tracking.
- **Theme Support**: 'Stadium', 'Classic', and 'Neon' modes.
